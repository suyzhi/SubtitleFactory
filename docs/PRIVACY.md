# 字幕工厂隐私说明

字幕工厂默认在本机处理媒体、音频、字幕、OCR、说话人识别、质检、术语表和翻译记忆。项目数据不会自动同步，也不会静默上传或在本地失败后自动回退云端。

YouTube 项目选择“网页播放”时，内嵌播放器会直接连接 YouTube 以加载和播放视频，因此 YouTube 可按其政策接收常规播放器请求与网络信息。转写音频、字幕、编辑历史和项目数据库仍保存在本机；切换为“下载至本地”后不再依赖网页播放器进行预览。

AI 翻译等联网能力只在用户配置 Provider 并主动执行后调用。OCR、说话人增强和 AI 质检分别保存首次授权记录；撤销后对应云端适配器必须停止调用。授权不等于上传，每次能力仍只处理用户主动提交的项目范围。

API Key 存入 macOS Keychain，SQLite 仅保留引用与“已配置”状态。诊断包不包含密钥、媒体、字幕正文或完整用户目录路径。

本地 API 使用每次启动随机令牌和动态端口，媒体链接按项目签名并会过期。普通网页不能读取项目数据或调用修改接口。
