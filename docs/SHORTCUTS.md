# 编辑器快捷键

| 操作 | 快捷键 |
| --- | --- |
| 播放 / 暂停 | `Space` |
| 前后一帧 / 小步移动 | `,` / `.` |
| 拆分当前字幕 | `Cmd/Ctrl + Enter` |
| 撤销 / 重做 | `Cmd/Ctrl + Z` / `Cmd/Ctrl + Shift + Z` |
| 保存当前草稿 | `Cmd/Ctrl + S` |
| 时间轴边界微调 | 聚焦边界后使用方向键 |
| 关闭弹层 | `Esc` |

时间码、时间轴边界、流程卡片和主要操作按钮均可通过键盘聚焦。系统开启“减少动态效果”时，应用会降低非必要动画。
